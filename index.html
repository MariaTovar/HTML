Qué es HTML, para qué funciona.

Html por sus siglas en inglés significa "Lenguaje de mercado de hipertexto". No es un lengua de programación, pues con HTML no creamos códigos complejos. Más bien, lo que nos permite hacer HTML, es "informarle" a un navegador web (google, rave, etc) qué elementos hay en la página. "Marca" el contenido, haciendo que el navegador sepa qué hay en la página y cómo estructurar la información que se está dando. Pudieramos decir que el navegador y la información son como cajas, y html nos permite organizar esas cajas según su tamaño, importancia, color, etc.

Qué son las etiquetas

Las etiquetas son los elementos que se utilizan para estructurar la información dentro de la página web. Componen el lenguaje de HTML. Con ellas el contenido agarra forma. Hay etiquetas para párrafos, para encabezados, para imágenes. Es la información plasmada ordenada a través de las etiquetas. Cada etiqueta tiene un propósito específico para dar estructura y presentación al contenido de la página web.

Qué son los atributos

Los atributos en HTML son elementos adicionales que se pueden agregar a una etiqueta para modificar o especificar ciertas características del elemento al que se aplican. Cuales son las etiquetas más importantes

A continuación algunas de las etiquetas más comunes y fundamentales de HTML

  • html: Define el comienzo y el final del documento
  • head: Contiene información meta sobre el documento, como el título, enlaces a hojas de estilo (CSS), y otros elementos meta.
  • title: Define el título del documento, que se muestra en la barra de título del navegador.
  • body: Contiene el contenido del documento, como texto, imágenes, enlaces, etc.
  • h1, h2, ..., h6: Definen encabezados de diferentes niveles, desde el más importante (h1) hasta el menos importante (h6).
  • p: Define un párrafo de texto.
  • a: Crea un enlace a otra página web o recurso.
  • img: Inserta una imagen en la página.
  • ul y ol`: Crean listas no ordenadas y ordenadas, respectivamente.
  • li: Define un elemento de lista dentro de ul o ol.
  • table: Crea una tabla en la página.
  • tr: Define una fila en una tabla.
  • td: Define una celda de datos en una tabla.
  • th: Define una celda de encabezado en una tabla.
  • form: Crea un formulario que puede contener campos de entrada, botones y otros elementos.
  • input: Define un campo de entrada en un formulario.
  • label: Etiqueta descriptiva para un elemento de formulario.
  • textarea: Crea un área de texto en un formulario.
  • select y option`: Crean listas desplegables en un formulario.
